# appCoinche

Server to client calls :
* Request : 
  * Status
  * Play a card
  * Make a bid
  * Update :
   * Score
   * Bid
   * Round
   * Trick
   * Hand
* Answer : Valid/Not valid

Client to server calls :
* Request resp :
  * Status : Waiting or Ready
  * Play a card : Card to play
  * Make a bid : Bid value
* See last trick
* Pause the game
